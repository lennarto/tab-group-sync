Tab Group Sync – Options

Sites

For Sheets, “bullet list” is treated as one-URL-per-line plain text so that pasting fills rows instead of a single rich-text cell.

Opening behavior

When enabled, newly added tabs will begin loading immediately even if they’re not selected.

Excluded hostnames

One pattern per line. Supports wildcards like *.google.com. Any link with a matching host is ignored by “Open all URLs” and “Copy new URLs”.

Tip: The options page itself is a chrome-extension:// URL and is always ignored.
